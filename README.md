# FlaskAuthApp
A simple Flask authentication project that allows users to register and log in.  
This project demonstrates basic user authentication using Python and Flask.

--------------------------------------------------------------------------------

# 🚀 Features
✔ User Registration  
✔ User Login  
✔ User Session Management  
✔ Template based UI (HTML)  
✔ Uses Flask routing and templates

--------------------------------------------------------------------------------

# 🛠️ Prerequisites
Make sure you have the following installed:

- Python 3.8+  
- pip (Python package installer)
